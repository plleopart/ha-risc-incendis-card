# Risc Incendis Card

Custom Lovelace card for Home Assistant to display the official Pla Alfa wildfire
risk level from the `risc_incendis` integration.

This card is designed to be used with:

- `sensor.pla_alfa_avui`
- `sensor.pla_alfa_dema`

or the entity IDs generated by Home Assistant for each configured municipality.

## Goal

Provide a visual, dashboard-friendly card for wildfire risk:

- large current Pla Alfa level
- dynamic color by level
- readable description
- municipality and comarca
- source update timestamp
- tomorrow preview when available
- graceful `unknown` / `S/D` state

## Usage

```yaml
type: custom:risc-incendis-card
entity: sensor.pla_alfa_avui
tomorrow_entity: sensor.pla_alfa_dema
title: Pla Alfa
```

If Home Assistant generated municipality-prefixed entity IDs, use those instead:

```yaml
type: custom:risc-incendis-card
entity: sensor.santa_maria_de_besora_pla_alfa_avui
tomorrow_entity: sensor.santa_maria_de_besora_pla_alfa_dema
```

## Options

| Option | Required | Description |
| --- | --- | --- |
| `entity` | yes | Sensor with today's Pla Alfa level. |
| `tomorrow_entity` | no | Sensor with tomorrow's Pla Alfa level. |
| `title` | no | Card title. Defaults to `Pla Alfa`. |
| `show_tomorrow` | no | Show tomorrow preview. Defaults to `true`. |
| `show_source` | no | Show source/update metadata. Defaults to `true`. |

## Visual Levels

| Level | Description | Color |
| --- | --- | --- |
| `0` | Perill baix | white / green |
| `1` | Perill moderat | yellow |
| `2` | Perill alt | orange |
| `3` | Perill molt alt | red |
| `4` | Perill extrem | dark red |
| `unknown` | Sense dades | gray |

## HACS Deployment

This repository is intended to be installed as a HACS **Plugin**.

Repository structure:

```text
risc-incendis-card.js
hacs.json
README.md
```

After installation, add the frontend resource in Home Assistant if HACS does not
do it automatically:

```yaml
url: /hacsfiles/ha-risc-incendis-card/risc-incendis-card.js
type: module
```

Then add the card to a dashboard using YAML mode or the manual card editor.

## Relationship With The Integration

This repository only contains the frontend Lovelace card.

The Home Assistant integration lives in:

```text
https://github.com/plleopart/ha-risc-incendis
```

Install the integration first, configure a municipality, and then use the created
sensors in this card.

## Development

During development, the card can be served from Home Assistant's `/config/www`
directory:

```text
/config/www/risc-incendis-card/risc-incendis-card.js
```

Resource URL:

```yaml
url: /local/risc-incendis-card/risc-incendis-card.js
type: module
```

Remember to clear the browser cache or change the resource URL query string
after each local build:

```yaml
url: /local/risc-incendis-card/risc-incendis-card.js?v=dev1
type: module
```
